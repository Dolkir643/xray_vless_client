# Xray VLESS Client

Аддон Home Assistant OS: клиент Xray по **VLESS**-ссылке (тот же формат, что в Shadowrocket / V2RayN). Поднимает локальный **SOCKS5**-прокси для обхода блокировок.

**Текущая версия: 1.0.2** — см. [CHANGELOG.md](CHANGELOG.md) (host_network, подключение по IP HA).

## Зачем

- Вставить вашу рабочую VLESS-ссылку из Shadowrocket — и пустить через неё трафик с HAOS (например, бота Telegram).
- Shadowrocket на HAOS поставить нельзя (это приложение под iOS/macOS), поэтому этот аддон заменяет его для сервера.

## Установка

1. Добавьте репозиторий в Home Assistant (если ещё не добавлен): **Настройки → Дополнения → Репозитории** — URL вашего репозитория с аддонами.
2. Обновите список дополнений и установите **«Xray VLESS Client»**.
3. В конфигурации укажите:
   - **vless_url** — ваша VLESS-ссылка целиком (как в Shadowrocket: `vless://uuid@host:port?type=tcp&security=reality&sni=...&pbk=...&...`).
   - **socks_port** — порт SOCKS5 (по умолчанию `1080`).
4. Запустите аддон.

## Использование с CarVector Bot

1. Запустите **Xray VLESS Client** и убедитесь, что он работает (в логах нет ошибок).
2. Узнайте **IP вашего Home Assistant** в локальной сети: **Настройки → Система → Сеть** (например `192.168.1.50`).
3. В аддоне **CarVector Bot** в поле **telegram_proxy** укажите:
   - **`socks5://ВАШ_IP_HA:1080`** — например `socks5://192.168.1.50:1080`.
   - Не используйте `172.17.0.1` — с других аддонов он часто недоступен; нужен именно IP HA в сети.
4. Сохраните конфиг и перезапустите CarVector Bot — запросы к Telegram пойдут через VLESS.

## Формат VLESS-ссылки

Ссылка вида:

```text
vless://UUID@сервер:порт?type=tcp&security=reality&sni=...&fp=chrome&pbk=...&sid=...
```

Поддерживаются:

- **transport:** `tcp`, `ws`, `grpc`, `http`
- **security:** `reality`, `tls`, без TLS
- Параметры Reality: `sni`, `fp`, `pbk`, `sid`, `flow` (например, `xtls-rprx-vision`)

Скопировать ссылку можно из Shadowrocket (долгое нажатие на конфиг → «Share» / «Скопировать ссылку»).

## Порты

- Аддон работает в режиме **host_network**: SOCKS5 слушает на порту **1080** прямо на хосте HA.
- Подключаться к прокси нужно по **IP вашего Home Assistant в сети** и порту 1080, например: `socks5://192.168.1.50:1080`.

## Безопасность

- VLESS-ссылку и параметры не логируем целиком.
- Прокси слушает на всех интерфейсах (`0.0.0.0`), доступ ограничивайте файрволом/сетью при необходимости.
