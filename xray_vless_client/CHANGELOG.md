# История изменений

## 1.0.2 (2025-03)

### Изменено
- **Режим host_network:** аддон теперь использует сеть хоста, SOCKS5 слушает напрямую на порту 1080 на вашем HA. Так к прокси стабильно могут подключаться другие аддоны (например CarVector Bot).
- В README явно указано: в **telegram_proxy** подставлять **IP вашего Home Assistant в локальной сети** (Настройки → Система → Сеть), например `socks5://192.168.1.50:1080`, а не `172.17.0.1`.

---

## 1.0.1 (2025-03)

### Исправлено
- **Поддержка VLESS-ссылок из Shadowrocket.** Раньше аддон ожидал только формат `vless://uuid@host:port?params`. Ссылки, скопированные из Shadowrocket, часто приходят в формате с **Base64**: часть после `vless://` закодирована (например `vless://YXV0bzpmOWJh...?peer=...&pbk=...`). Теперь такой формат корректно декодируется и разбирается.
- Ошибка «Формат: vless://uuid@host:port» при вставке ссылки из Shadowrocket больше не возникает.

### Добавлено
- Поддержка параметров Shadowrocket: `peer` (SNI), `tls`, `xtls`, `pbk`, `sid` — автоматическое определение Reality/TLS и подстановка в конфиг Xray.
- В логах при старте выводится версия аддона: `Xray VLESS Client v1.0.1`, чтобы было видно, что установлена обновлённая версия.
